# Pipeline Hackathon Rules

Additional rules for hackathon sprint mode (24h). These EXTEND hp-core rules.

## Time Management

1. **Phase timing is sacred** — Don't let any phase run over. Hard stops.
2. **Brainstorm cap: 2.5h** — If no decision by T+2:30, pick simplest viable idea
3. **Mid-sprint QA** — Check at halfway point of each build sprint
4. **Test deploy early** — Deploy after Sprint 1, not at the end. Catch deploy issues early.
5. **Presentation is NOT optional** — 33% of score. Allocate full 2.5h.

## Innovation Focus

6. **AI is core, not decorative** — AI must DRIVE the experience. If it could work without AI, rethink.
7. **Novel combinations win** — New combinations of existing tech > new tech from scratch
8. **Show AI reasoning** — Make AI tool use and reasoning visible to judges
9. **Feedback loops matter** — User → AI → Action → User is more impressive than one-shot AI
10. **The pipeline itself is an innovation** — Consider showing how you built it

## Demo-First Development

11. **Build for the demo** — If it won't be shown in 5 minutes, deprioritize
12. **Golden path** — Ensure one complete flow works perfectly. Polish that path.
13. **Demo data matters** — Seed compelling, realistic data that tells a story
14. **Fallback plan required** — Screenshots, pre-recorded video, pre-cached AI responses
15. **Rehearse** — Practice the exact demo 2-3 times with a timer

## Quality Over Quantity

16. **3 polished features > 10 half-broken features**
17. **Working demo > beautiful non-working app**
18. **Depth over breadth** — Go deep on core feature, shallow on extras
19. **No dead UI** — If a button exists, it must do something

## Sustainability Theme (Adria Future Hackathon)

20. **Real data** — Use actual environmental/sustainability data where possible
21. **Local relevance** — Western Balkans specificity differentiates from generic solutions
22. **Impact numbers** — Quantify sustainability impact, even if projected
23. **Visual narrative** — Earth tones, nature imagery, clean design for sustainability theme
