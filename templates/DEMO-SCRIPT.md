## Demo Flow

### [00:00 — 00:20] Opening
- **Action:** Show landing/home page
- **Expected:** [what should be visible]

### [00:20 — 00:50] Core Feature
- **Action:** [Click specific element / navigate to specific page]
- **Expected:** [What should appear on screen]
- **Say:** "Watch how [feature] works in practice..."
- **WAIT:** for [specific element/data] to load before continuing
- **Transition:** [next action]

### [00:50 — 01:20] AI in Action
- **Action:** [Type specific prompt: "exact text to type"]
- **Expected:** [AI response with visible reasoning/tool use]
- **Say:** "The AI is analyzing [data source] in real-time. Notice how it [specific capability]..."

### [01:20 — 01:50] Impact / Results
- **Action:** [Navigate to results/dashboard/visualization]
- **Expected:** [Data visualization or impact display]

### [01:50 — 02:00] Wrap
- **Action:** Return to landing page or show summary view

## Fallback Plans

| Scenario | Action |
|----------|--------|
| AI response is slow (>10s) | Say "While that processes..." and show pre-computed results |
| App crashes | Switch to presentation slides with screenshots |
| Network is down | Show pre-recorded video from presentation/ |
| Demo data missing | Use backup dataset (instructions: [how to load]) |
| Feature doesn't work | Skip to next section, mention "We also built..." |
| Projector resolution wrong | Zoom browser to 125% or 150% |
